# Sudoku
Play sudoku on a webapp

### Project docs
[Check out the docs!](https://github.com/BBence110/Sudoku-mini-project-CSK4/tree/master/docs)

### Prerequisites
- Java 21
- Spring Boot 3.3.2
- Spring boot AOP
- Maven latest
- H2 (With the dev profile you can use H2)
- tomcat

### Getting Started
##### Deployment Process

Development start:
```sh
mvn clean install
mvn spring-boot:run
```

#####  Application :
- http://localhost:9090/
