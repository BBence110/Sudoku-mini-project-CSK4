# Sudoku
Play sudoku on a webapp

### Prerequisites
- JSF 2.2.9
- Java 8
- Spring Boot 2.0.0.RELEASE 
- Spring boot AOP
- Maven latest
- H2 (With the dev profile you can use H2)
- tomcat

### Getting Started
##### Deployment Process

Development start:
```sh
mvn clean install
mvn spring-boot:run
```

#####  Application :
- http://localhost:9090/
